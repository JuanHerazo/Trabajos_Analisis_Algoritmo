### Ejercicio 1 – LeetCode: Valid Anagram

**Enunciado:**  
Dadas dos cadenas `s` y `t`, devolver `true` si `t` es un anagrama de `s`, y `false` en caso contrario. Un anagrama es una cadena que se forma reordenando las letras de otra (mismas letras y mismas frecuencias).

**Ejemplo:**
- Entrada: `s = "anagram"`, `t = "nagaram"` → Salida: `true`
- Entrada: `s = "rat"`, `t = "car"` → Salida: `false`

**URL:** https://leetcode.com/problems/valid-anagram/

---

### Ejercicio 2 – LeetCode: Contains Duplicate

**Enunciado:**  
Dado un arreglo de enteros `nums`, devolver `true` si algún valor aparece al menos dos veces en el arreglo, y `false` si todos los elementos son distintos.

**Ejemplo:**
- Entrada: `nums = [1, 2, 3, 1]` → Salida: `true`
- Entrada: `nums = [1, 2, 3, 4]` → Salida: `false`

**URL:** https://leetcode.com/problems/contains-duplicate/

---

### Ejercicio 3 – LeetCode: Best Time to Buy and Sell Stock

**Enunciado:**  
Dado un arreglo `prices` donde `prices[i]` es el precio de una acción en el día `i`, devolver la **máxima ganancia** posible eligiendo un día para comprar y un día posterior para vender. Si no es posible obtener ganancia, devolver `0`.

**Ejemplo:**
- Entrada: `prices = [7, 1, 5, 3, 6, 4]` → Salida: `5` (comprar en 1, vender en 6)
- Entrada: `prices = [7, 6, 4, 3, 1]` → Salida: `0`

**URL:** https://leetcode.com/problems/best-time-to-buy-and-sell-stock/

---

### Ejercicio 4 – Codeforces: Stones on the Table (266A)

**Enunciado:**  
Hay `n` piedras en una fila, cada una de color rojo (R), verde (G) o azul (B). Calcular el **mínimo número de piedras** que hay que quitar de la fila para que **no queden dos piedras adyacentes del mismo color**.

**Entrada:** Primera línea un entero `n`. Segunda línea una cadena de `n` caracteres (solo R, G, B).  
**Salida:** Un entero con el mínimo de piedras a quitar.

**Ejemplo:**
- Entrada: `3` y `RRG` → Salida: `1`
- Entrada: `5` y `RRRRR` → Salida: `4`

**URL:** https://codeforces.com/problemset/problem/266/A
